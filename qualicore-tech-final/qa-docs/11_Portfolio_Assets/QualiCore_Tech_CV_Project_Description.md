## Full Version (for project section with bullet points)

**QualiCore Tech — Manual QA Portfolio Project**  
*Solo project | React + Vite e-commerce application | March 2026*

- Planned and executed a full manual QA cycle on a realistic demo e-commerce application (50+ products, 13 routes, cart, wishlist, auth, checkout)
- Reviewed Software Requirements Specification and produced 50 user stories across 9 functional epics
- Designed 37 test scenarios and 35 detailed test cases covering all major functional areas
- Maintained a Requirements Traceability Matrix linking requirements, user stories, acceptance criteria, test scenarios, and test cases
- Executed all 35 test cases and achieved an end-to-end pass rate of 62.9% (22/35)
- Identified and documented 15 defects including: cart subtotal calculation error, search result inconsistency, sorting logic defect (components category), missing form validation (login, checkout), broken product images, and data consistency issues
- Wrote all bug reports in professional Jira-style format with severity/priority ratings, full reproduction steps, and impact assessment
- Produced a Test Summary Report with quality assessment, pass/fail breakdown by area, defect severity distribution, and release recommendation
- Application deployed to Vercel as a live, interactive QA practice environment

**Tech tested:** React 18 · React Router · localStorage · Tailwind CSS · Chrome DevTools (mobile simulation)  
**QA artefacts:** SRS · User Stories · Acceptance Criteria · RTM · Test Plan · Test Scenarios · Test Cases · Bug Reports · Execution Log · Test Summary Report
