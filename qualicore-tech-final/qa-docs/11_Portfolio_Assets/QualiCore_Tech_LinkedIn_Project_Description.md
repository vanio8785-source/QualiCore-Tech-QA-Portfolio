QualiCore Tech is a full manual QA portfolio project I built and tested end-to-end to demonstrate real-world software testing skills.

🧪 **Testing a realistic e-commerce web application** built with React, covering 13 pages and features including product search, filtering, sorting, cart management, wishlist, user authentication, and checkout.

📋 **Full QA documentation suite** produced from scratch:
- Software Requirements Specification (SRS)
- 50 User Stories across 9 functional epics
- Acceptance Criteria for all major features
- Requirements Traceability Matrix (RTM)
- QA Test Plan
- 37 Test Scenarios
- 35 Detailed Manual Test Cases
- 15 Professional Bug Reports
- Test Execution Log
- Test Summary Report

🐛 **15 defects discovered and documented**, including:
- Cart subtotal calculation error on a specific product
- Sorting logic defect affecting an entire product category
- Form validation gaps in login, registration, and checkout
- Search result inconsistency for a specific brand
- Product data discrepancies (broken images, incorrect specs, page title typo)
- Wishlist data loss on page refresh
- Cart quantity exceeding stock limits

📊 **Test execution results:** 35 test cases executed · 22 passed · 13 failed · 62.9% pass rate

This project simulates the full QA lifecycle I would follow on a real product team — from requirements review through test design, execution, defect reporting, and summary reporting.
