# QualiCore Tech — User Stories v1.0

## Product Browsing

**US-001**: As a user, I want to view all products on the catalog page so I can browse available items.

**US-002**: As a user, I want to click on a product to view its full details, specifications, and images.

**US-003**: As a user, I want to see the product price, rating, and availability clearly on each product card.

**US-004**: As a user, I want to see sale/discount badges on discounted products.

## Search

**US-005**: As a user, I want to search for products by name so I can find specific items quickly.

**US-006**: As a user, I want search to match across product name, brand, and category.

**US-007**: As a user, I want to see a helpful message when my search returns no results.

## Filtering

**US-008**: As a user, I want to filter products by category so I can narrow the catalog.

**US-009**: As a user, I want to filter products by brand.

**US-010**: As a user, I want to filter products by price range.

**US-011**: As a user, I want to filter by availability (in-stock only).

**US-012**: As a user, I want to clear all filters and see the full catalog again.

## Sorting

**US-013**: As a user, I want to sort products by price (low to high or high to low).

**US-014**: As a user, I want to sort products by name alphabetically.

**US-015**: As a user, I want to sort products by rating.

## Cart

**US-016**: As a user, I want to add a product to my cart from the catalog or product detail page.

**US-017**: As a user, I want to view my cart with all added items, quantities, and prices.

**US-018**: As a user, I want to change the quantity of items in my cart.

**US-019**: As a user, I want to remove items from my cart.

**US-020**: As a user, I want my cart to persist if I refresh the page.

**US-021**: As a user, I want to see a cart badge in the header showing how many items I have.

## Wishlist

**US-022**: As a user, I want to add products to my wishlist for later.

**US-023**: As a user, I want to view my wishlist page.

**US-024**: As a user, I want to remove items from my wishlist.

**US-025**: As a user, I want my wishlist to persist across page refreshes.

## Authentication

**US-026**: As a user, I want to log in with demo credentials.

**US-027**: As a user, I want to register a new account.

**US-028**: As a user, I want to log out.

## Checkout

**US-029**: As a user, I want to proceed to checkout from my cart.

**US-030**: As a user, I want to fill in my delivery and payment details.

**US-031**: As a user, I want required fields to be validated before submission.

**US-032**: As a user, I want to see an order summary during checkout.

**US-033**: As a user, I want to see an order confirmation after placing my order.

**US-034**: As a user, I want free shipping on orders over £50.

## Responsive & Navigation

**US-035**: As a user, I want the site to be usable on mobile devices.

**US-036**: As a user, I want to navigate between pages using the header menu.

**US-037**: As a user, I want to access the About, Contact, and Terms pages from the footer.
