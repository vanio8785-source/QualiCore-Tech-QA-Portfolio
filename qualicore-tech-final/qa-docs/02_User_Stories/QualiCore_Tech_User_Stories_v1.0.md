# User Stories
## QualiCore Tech — Manual QA Portfolio E-Commerce Application
**Document Version:** 1.0  
**Date:** March 2026

---

## Epic 1 — Product Browsing

**US-001**  
As a guest user, I want to view the home page so that I can get an overview of the store and navigate to product areas.

**US-002**  
As a guest user, I want to browse all products in a catalogue so that I can explore what the store offers.

**US-003**  
As a guest user, I want to view a product detail page so that I can see full product information before adding to cart.

**US-004**  
As a guest user, I want to see product images and a gallery so that I can evaluate the product visually.

**US-005**  
As a guest user, I want to see a product's specifications table so that I can compare technical details.

**US-006**  
As a guest user, I want to see the product's price, any discount, and the old price so that I understand the value I am getting.

**US-007**  
As a guest user, I want to see the product's stock status so that I know whether it is available to purchase.

**US-008**  
As a guest user, I want to see related products at the bottom of a product page so that I can discover alternatives.

---

## Epic 2 — Search

**US-009**  
As a guest user, I want to search for products using the header search bar so that I can quickly navigate to what I'm looking for.

**US-010**  
As a guest user, I want to search for products using the sidebar search on the catalogue page so that I can filter results inline.

**US-011**  
As a guest user, I want to search by product name, brand, category, or description so that I can find items using different terms.

**US-012**  
As a guest user, I want the search results count to update so that I know how many products matched my search.

---

## Epic 3 — Filtering

**US-013**  
As a guest user, I want to filter products by category so that I can narrow down results to a specific product type.

**US-014**  
As a guest user, I want to filter products by brand so that I can find products from a preferred manufacturer.

**US-015**  
As a guest user, I want to filter products by a maximum price so that I can shop within my budget.

**US-016**  
As a guest user, I want to filter products by minimum rating so that I can find well-reviewed items.

**US-017**  
As a guest user, I want to filter by in-stock only so that I do not see unavailable products.

**US-018**  
As a guest user, I want to clear all active filters with a single click so that I can reset the catalogue view quickly.

---

## Epic 4 — Sorting

**US-019**  
As a guest user, I want to sort products by price low to high so that I can find the most affordable options first.

**US-020**  
As a guest user, I want to sort products by price high to low so that I can find premium products first.

**US-021**  
As a guest user, I want to sort products by name A–Z or Z–A so that I can locate products alphabetically.

**US-022**  
As a guest user, I want to sort products by best rated so that I can find top-reviewed products easily.

**US-023**  
As a guest user, I want the default sort order to show newest products first so that I see the latest additions.

---

## Epic 5 — Cart

**US-024**  
As a guest user, I want to add a product to my cart from the product listing so that I can shop without entering each product page.

**US-025**  
As a guest user, I want to add a product to my cart from the product detail page with a selected quantity so that I can control how many units I add.

**US-026**  
As a guest user, I want to view my cart and see all added items, quantities, prices, and totals so that I can review my order.

**US-027**  
As a guest user, I want to increase or decrease item quantities in the cart so that I can adjust my order.

**US-028**  
As a guest user, I want to remove an item from the cart so that I can change my selection.

**US-029**  
As a guest user, I want to see the subtotal, shipping cost, and total in the cart so that I understand what I will pay.

**US-030**  
As a guest user, I want to see a free shipping notification when my order qualifies so that I understand the shipping benefit.

**US-031**  
As a guest user, I want my cart to persist across page refreshes so that I do not lose my selections.

---

## Epic 6 — Wishlist

**US-032**  
As a guest user, I want to add a product to my wishlist so that I can save it for later consideration.

**US-033**  
As a guest user, I want to remove a product from my wishlist so that I can manage my saved items.

**US-034**  
As a guest user, I want to view all items in my wishlist on a dedicated page so that I can review my saved products.

**US-035**  
As a guest user, I want the wishlist badge count in the header to reflect my current wishlist size so that I always know how many items I have saved.

---

## Epic 7 — Authentication

**US-036**  
As a guest user, I want to log in using valid credentials so that I can access my account.

**US-037**  
As a guest user, I want to see an error message if I submit incorrect login credentials so that I understand why login failed.

**US-038**  
As a guest user, I want to register a new account so that I can become an authenticated user.

**US-039**  
As an authenticated user, I want to see my name displayed in the header so that I know I am logged in.

**US-040**  
As an authenticated user, I want to log out so that I can end my session.

---

## Epic 8 — Checkout

**US-041**  
As a user with items in my cart, I want to proceed to checkout so that I can complete my purchase.

**US-042**  
As a user, I want to enter my delivery details during checkout so that my order can be delivered.

**US-043**  
As a user, I want to enter payment details during checkout so that I can complete the purchase.

**US-044**  
As a user, I want required fields to be validated during checkout so that I cannot submit an incomplete order.

**US-045**  
As a user, I want to see an order summary during checkout so that I can confirm my items and total before placing the order.

**US-046**  
As a user, I want to be redirected to an order confirmation page after placing an order so that I receive confirmation of my purchase.

---

## Epic 9 — Navigation & Responsive Behaviour

**US-047**  
As a mobile user, I want to open and close a mobile navigation menu so that I can navigate the site from a small screen.

**US-048**  
As a mobile user, I want the layout to adapt to my screen size so that the content is readable and usable.

**US-049**  
As a user, I want the header navigation links to take me to the correct pages so that I can navigate efficiently.

**US-050**  
As a user, I want breadcrumb navigation on the product detail page so that I can understand my location and navigate back.
