# QualiCore Tech – Portfolio Submission Guide

## Project Overview
QualiCore Tech is a realistic demo e-commerce web application created for manual QA practice and portfolio demonstration.

The project simulates a consumer electronics online store and includes core storefront functionality such as product browsing, product details, cart, wishlist, authentication flows, checkout, and responsive behavior.

This application was intentionally designed as a QA training environment and includes a controlled set of reproducible defects for testing practice.

---

## Project Purpose
The main goal of this project is to provide a realistic environment for practicing and demonstrating:

- Functional Testing
- Exploratory Testing
- Regression Testing
- UI / UX Validation
- Negative Testing
- Bug Reporting
- Test Documentation
- Requirement Traceability

The project is intended to support the development of practical QA skills through hands-on testing of a modern web application.

## How to Run the Application

### Prerequisites
- Node.js 18 or later installed
- npm (comes with Node.js)

### Steps

```bash
# Navigate to the app folder
cd qualicore-tech-final/app

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at: **http://localhost:5173**

### Other commands

```bash
# Build for production
npm run build

# Preview production build locally
npm run preview
```
